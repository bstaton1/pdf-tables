# Example Table with Select Cells Shaded

This example uses `knitr::kable() %>% kableExtra::funs()` to produce a publication-quality table that includes selected cells shaded.

This example showcases these features within the `knitr::kable() %>% kableExtra::funs()` framework:

* LaTeX math environments
* Shading of select cells
* Grouped columns
* Multi-line cells

The content in this example is part of a supplement to a manuscript currently in preparation.

_This example was submitted for the [2020 RStudio Table Contest](https://blog.rstudio.com/2020/09/15/announcing-the-2020-rstudio-table-contest/)._
