# Example Large Multi-page Table

This example uses `knitr::kable() %>% kableExtra::funs()` to produce a publication-quality table that spans multiple pages and has a relatively complex footnote section.

This example showcases these features within the `knitr::kable() %>% kableExtra::funs()` framework:

- Multi-page table
- Multi-line cells
- Grouped rows
- Grouped columns
- Multi-line cells
- Fairly detailed footnotes section

The content in this example was adapted from:

> Staton, B.A. 2018. In-season harvest and effort estimates for 2018 Kuskokwim River subsistence salmon fisheries during block openers. Project Summary Report. Yukon Delta National Wildlife Refuge, U.S. Fish and Wildlife Service. Available from: <https://www.fws.gov/uploadedFiles/2018KuskokwimRiverSalmonSubsistenceHarvestReport.pdf>
